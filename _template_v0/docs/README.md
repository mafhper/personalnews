# Documentação do Projeto (docs/)

## Propósito

Este diretório contém toda a documentação relevante para o projeto AuraWall. Desde guias técnicos, logs de mudanças até insights de conhecimento. É a "memória" do projeto, servindo como referência para desenvolvedores e colaboradores.

## Conteúdo

### Arquivos
- `change.log`: Registro cronológico de todas as mudanças e marcos importantes do projeto.
- `TECHNICAL_GUIDE.md`: Um guia técnico aprofundado que descreve a arquitetura, stack tecnológica, padrões de desenvolvimento e processos do projeto.
