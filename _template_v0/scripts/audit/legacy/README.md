# Scripts de Auditoria Legados (scripts/audit/legacy/)

## Propósito

Este diretório armazena scripts de auditoria mais antigos que foram substituídos por soluções mais modernas e unificadas (principalmente o `audit/runner.cjs`). Eles são mantidos aqui por razões históricas ou de compatibilidade, mas são considerados depreciados e não devem ser usados em novos fluxos de trabalho. Serão removidos em futuras versões.

## Conteúdo

### Arquivos
- `app.cjs`: Script de auditoria legado específico para a aplicação principal.
- `landing.cjs`: Script de auditoria legado específico para a landing page (site promocional).
