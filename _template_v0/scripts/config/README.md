# Configurações de Scripts (scripts/config/)

## Propósito

Este diretório contém arquivos de configuração que são utilizados pelos scripts do projeto. Centraliza parâmetros e opções que podem ser ajustados sem a necessidade de modificar a lógica principal dos scripts.

## Conteúdo

### Arquivos
- `audit.config.cjs`: Configurações globais e definições de "targets" (URLs e características) para os scripts de auditoria de performance, como o `audit/runner.cjs`.
