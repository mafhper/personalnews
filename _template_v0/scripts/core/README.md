# Módulos Core dos Scripts (scripts/core/)

## Propósito

Este diretório contém módulos de funcionalidades "core" que são compartilhados e reutilizados por diversos outros scripts de automação. Promove a modularidade e evita a duplicação de código em tarefas comuns.

## Conteúdo

### Arquivos
- `run-audit.cjs`: Módulo que encapsula a lógica central de execução de uma auditoria Lighthouse para um target específico, incluindo a inicialização de servidor e processamento de resultados.
