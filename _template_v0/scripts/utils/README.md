# Funções Utilitárias de Scripts (scripts/utils/)

## Propósito

Este diretório contém funções utilitárias e módulos auxiliares que são projetados para serem reutilizados por diversos scripts de automação. Eles fornecem funcionalidades comuns, como manipulação de arquivos de relatório, formatação e logging, promovendo a consistência e a manutenção dos scripts.

## Conteúdo

### Arquivos
- `audit-helpers.cjs`: Funções auxiliares para scripts de auditoria, como salvar relatórios em formatos padronizados (Markdown, JSON) e organizar os diretórios de saída.
