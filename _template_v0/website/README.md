# Site Promocional (website/)

## Propósito

Este diretório contém o código fonte e os recursos para o site promocional do AuraWall. É um projeto Vite separado, configurado para Server-Side Rendering (SSR) e Static Site Generation (SSG), destinado a apresentar o produto, documentar suas funcionalidades e servir como ponto de entrada público.

## Conteúdo

### Subdiretórios
- `public/`: Assets estáticos específicos do site promocional (imagens, favicon, etc.).
- `src/`: Código fonte da aplicação React do site promocional.

### Arquivos
- `vite.config.ts`: Configuração do Vite para o build do site promocional.
- `package.json`: (Não listado explicitamente, mas é comum ter um para projetos separados se não for monorepo)
