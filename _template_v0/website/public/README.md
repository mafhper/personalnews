# Assets Públicos do Site Promocional (website/public/)

## Propósito

Este diretório contém os assets estáticos que são específicos para o site promocional. Estes arquivos são servidos diretamente pelo servidor web do site promocional e incluem páginas pré-renderizadas, imagens, favicons e dados como o changelog.

## Conteúdo

### Arquivos
- `404.html`: Página de erro 404 pré-renderizada para o site.
- `architecture-diagram.jpg`: Diagrama visual da arquitetura do projeto.
- `bg-boreal.svg`, `bg-chroma.svg`, etc.: Arquivos SVG de background estáticos utilizados em seções do site promocional.
- `changelog.json`: Dados do histórico de mudanças do projeto, gerados automaticamente por script.
- `favicon.ico`: Favicon padrão do site promocional.
- `header-animation.svg`: SVG utilizado para animações no cabeçalho.
- `icon-light.svg`: Ícone da aplicação em versão clara, usado no site.
