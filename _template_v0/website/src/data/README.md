# Dados Estáticos do Site Promocional (website/src/data/)

## Propósito

Este diretório armazena dados estáticos ou pré-definidos que são utilizados pelo site promocional. Isso inclui metadados e configurações que não são dinâmicas ou que não vêm de uma API externa.

## Conteúdo

### Arquivos
- `engines.ts`: Definições de metadados para os engines de geração de wallpaper, utilizados para exibir informações sobre cada motor no site promocional.
