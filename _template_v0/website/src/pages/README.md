# Páginas do Site Promocional (website/src/pages/)

## Propósito

Este diretório contém os componentes React que definem as diferentes páginas do site promocional. Cada arquivo `*.tsx` geralmente corresponde a uma rota ou visualização principal acessível pelo usuário.

## Conteúdo

### Arquivos
- `About.tsx`: Página "Sobre" o projeto AuraWall.
- `Changes.tsx`: Página que exibe o histórico de mudanças (changelog) do projeto.
- `Creation.tsx`: Página de hub para a seção de criação.
- `CreationAnimation.tsx`: Página detalhando os aspectos de animação na criação de wallpapers.
- `CreationBoreal.tsx`: Página (potencialmente legada ou específica) do motor Boreal.
- `CreationChroma.tsx`: Página (potencialmente legada ou específica) do motor Chroma.
- `CreationEngineDetail.tsx`: Página de detalhes dinâmica para um motor de geração específico.
- `CreationEngines.tsx`: Página que lista e apresenta todos os motores de geração disponíveis.
- `CreationProcedural.tsx`: Página detalhando os aspectos procedurais na criação de wallpapers.
- `Home.tsx`: A página inicial (landing page) do site promocional.
- `NotFound.tsx`: Página exibida quando uma rota não é encontrada (erro 404).
- `Tech.tsx`: Página que apresenta o stack tecnológico e aspectos de arquitetura do projeto.
