# Funções Utilitárias do Site Promocional (website/src/utils/)

## Propósito

Este diretório contém funções auxiliares e utilitários específicos para as operações e funcionalidades do site promocional. Estas funções ajudam a manter o código das páginas e componentes mais limpo e focado em sua lógica principal.

## Conteúdo

### Arquivos
- `appUrl.ts`: Utilitário para gerar URLs para a aplicação principal (`getAppUrl`), garantindo que os links para o editor ou presets funcionem corretamente, especialmente em ambientes de deploy (como GitHub Pages).
