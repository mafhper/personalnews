# Icon Package - Dashboard | Quality Core

Generated with Icon Forge

## Contents

- `light/` - Light theme icons
- `dark/` - Dark theme icons

Each variant folder contains:
- `web/` - Browser favicons
- `apple/` - Apple touch icons
- `pwa/` - Progressive Web App icons
- `social/` - Open Graph and Twitter card images
- `windows/` - Windows tiles and app icons
- `macos/` - macOS dock icons
- `linux/` - Linux desktop icons
- `manifest.json` - PWA manifest file
- `implementation.html` - Example HTML with all link tags

## Installation

1. Copy the icon files to your project's public folder
2. Add the link tags from `implementation.html` to your HTML head
3. Update the manifest.json with your app's information
